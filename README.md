# Arduino FSM & UART LED Controller

This project demonstrates Finite State Machine (FSM) based LED control using Arduino.

## Features
- FSM-based LED states (steady, slow blink, fast blink, off)
- Non-blocking timing using millis()
- UART control using Serial Monitor

## Files
- fsm_led_controller_v1.ino
- fsm_led_controller_v2.ino
- UART_CONTROLLED_LED.ino

## Concepts Used
- FSM
- UART (Serial Communication)
- millis() timing
